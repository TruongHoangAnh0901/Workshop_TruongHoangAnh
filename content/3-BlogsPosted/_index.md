This section lists and introduces the technical AWS blogs I have researched and read during the implementation of the Serverless Quiz App project.

###  [Blog 1 - Exploring serverless patterns for Amazon DynamoDB](3.1-Blog1/)
This blog post from the AWS Compute Blog dives deep into common serverless architectural patterns when working with DynamoDB. From connecting API Gateway directly to DynamoDB, to using Lambda as an intermediary business logic layer, the article provides best practices for optimizing performance and cost.

###  [Blog 2 - Amazon CloudFront introduces Origin Access Control (OAC)](3.2-Blog2/)
The official article from AWS about the launch of the Origin Access Control (OAC) security feature. It explains why OAC is the new standard replacing the old OAI, offering stronger security (preventing the Confused Deputy problem) and comprehensive support for HTTP methods to protect your S3 Bucket.

###  [Blog 3 - AWS AI: Artificial Intelligence on AWS](3.3-Blog3/)
This post dives deep into core AWS AI/ML services (like Amazon SageMaker, Amazon Rekognition, Amazon Transcribe) to highlight how AWS "democratizes" the power of Artificial Intelligence and Machine Learning, making it accessible for any organization to apply in reality.
