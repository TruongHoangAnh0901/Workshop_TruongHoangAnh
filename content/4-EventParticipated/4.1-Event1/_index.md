# Event: Automated Prompt Engineering: Enhancing LLM Output Quality Training

**Date & Time:** 09:00 - 12:00, May 09, 2026  
**Location:** 26th Floor, Bitexco Tower, 02 Hai Trieu Street, Saigon Ward, Ho Chi Minh City  
**Role:** Attendee  

### Event Objectives
- Understand the importance of Prompt Engineering in optimizing LLM output and cost savings (Token Economics).
- Grasp the standard structure of an effective Prompt and advanced techniques.
- Explore the real-world system architecture of an AI-integrated application ("Proptimizer" project) on the AWS platform.

### Speaker
- **Nguyen Tuan Thinh** – DevOps/Cloud Engineer, First Cloud AI Journey

### Key Highlights

#### The Art of Communicating with AI
- **Problem Statement:** Using generic prompts leads to wasted tokens, increased costs, and inconsistent results.
- **Anatomy of a Great Prompt:** Requires a clear structure including: Role, Instruction, Context, Input Data, Output Format, Examples, and Constraints.
- **Best Practices:** Use clear directional language, utilize delimiters, focus on what "TO DO" (DOs) instead of what "NOT TO DO" (DON'Ts), and break down overly long inputs.

#### Advanced Techniques & Real-world Application
- **Advanced Prompting:** Introduction and comparison of thinking models such as Chain-of-Thought (CoT), Self-Consistency, Tree-of-Thoughts (ToT), and Retrieval-Augmented Generation (RAG).
- **Proptimizer Project:** Analysis of a real-world application as a browser extension that helps optimize prompts.
- **Solution Architecture on AWS:** Discovering the cloud infrastructure for an AI application, from Frontend (CloudFront, S3), Authentication (Cognito), Backend Logic (API Gateway, Lambda), AI Integration (Bedrock), NoSQL Storage (DynamoDB), to System Monitoring (CloudWatch).

### Event Experience

Participating in the "Automated Prompt Engineering" training provided me with a comprehensive perspective on both AI communication skills and how to build a real-world AI-integrated application. Specific values gained include:

- **Mastering AI Communication:** Learned how to systematically refine LLM prompts. Applying techniques like Chain-of-Thought (CoT) or Role Prompting significantly improves performance and accuracy when using AI to solve problems.
- **Backend & Cloud Architecture Exposure:** Analyzing the Proptimizer project's architecture provided highly valuable practical knowledge. Understood how AWS Lambda and Amazon API Gateway play a role in routing and processing serverless logic, as well as how to connect with Amazon Bedrock to invoke foundation AI models.
- **Data & Security Awareness:** Grasped the user authentication flow using JWT tokens via Amazon Cognito and high-speed data storage methods (millisecond response times) using NoSQL databases like DynamoDB.
- **Productivity Boost:** Gained a clear awareness of "Token Economics" to design cost-optimized application processing flows. These skills can be directly applied to current workflows to enhance productivity and code quality.
