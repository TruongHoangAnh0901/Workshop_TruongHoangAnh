**On this page**, you will need to introduce your worklog. **How** did you complete it? How many weeks did you take to complete the program? **What** did you do in those weeks?

Typically, and as a standard, a worklog is carried out over about 3 months (throughout the internship period) with weekly contents as follows:

**Week 1:** [Foundations of AWS Cloud (Identity, Compute & Load Balancing)](1.1-week1/)

**Week 2:** [Cloud Observability & Centralized Account Governance](1.2-week2/)

**Week 3:** [Hybrid Storage Solutions, Web Application Firewall & Resource Management](1.3-week3/)

**Week 4:** [Real-world Implementation: Architecting a Scalable Web Infrastructure](1.4-week4/)

**Week 5:** [Enhancing Web Apps: Object Storage, Security Hardening & IaC Basics](1.5-week5/)

**Week 6:** [Infrastructure as Code Mastery using AWS CloudFormation](1.6-week6/)

**Week 7:** [Serverless Paradigms & Introduction to Terraform](1.7-week7/)

**Week 8:** [Advanced Terraform & Automating Deployments via AWS CI/CD](1.8-week8/)

**Week 9:** [Deep Dive into DevOps Tooling (Jenkins, Docker & Deployment Strategies)](1.9-week9/)

**Week 10:** [Getting Started with Container Orchestration (Kubernetes) & Disaster Recovery](1.10-week10/)

**Week 11:** [Kubernetes at Scale: Traffic Routing, Security Policies & Auto-scaling](1.11-week11/)

**Week 12:** [High-Performance Networking & Load Testing Methodologies](1.12-week12/)
