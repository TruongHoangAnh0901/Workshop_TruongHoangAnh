**Tuần 4:** [Triển khai thực tế: Xây dựng Kiến trúc Web linh hoạt (VPC, EC2)](1.4-week4/)

**Tuần 5:** [Nâng cấp Web App: Tích hợp lưu trữ S3, Bảo mật & Cơ bản về IaC](1.5-week5/)

**Tuần 6:** [Làm chủ Infrastructure as Code (IaC) với AWS CloudFormation](1.6-week6/)
